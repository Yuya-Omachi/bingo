# 使い方
python環境上でmain.pyを実行してください。

## 必要なライブラリ
numpy==1.24.2

# 表示内容
ボールを1回取り出すたびにボールのナンバー、現在の盤面、リーチの数、ビンゴの数が出力されます。表示例は以下の通りです。
```
ball[0]: 13
 05   16   31   51   62
 06   17   34   53   63
 08   20  FREE  54   66
 11   24   43   55   68
(13)  25   44   56   69

REACH: 0
BINGO: 0
```
丸括弧で囲まれた数字は開いているマスを示します。
